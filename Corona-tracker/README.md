✔ CSS <br>
✔ JavaScript <br>
✔ NodeJs <br>
✔ ExpressJs <br>
</p>
<h1 align="center"> 🌎 Use</h1>
- Download or clone the repository

```
git clone https://github.com/Ayushparikh-code/Web-dev-mini-projects.git
```

- Go to the directory
```
cd Corona-tracker
```

- Run the command
```
npm start
```
- Start Local sever at 1338
```
http://localhost:1338
```
